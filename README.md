Doctor Appointment System (backend + client added)

This archive contains:
- existing backend files (assumed present in your workspace)
- a new `client` React app in the `client` folder.

To use:
1. In the project root (where server.js and package.json exist), run:
   npm install
2. Install client deps:
   cd client
   npm install
3. Start backend and client concurrently (project package.json already has scripts):
   cd .. (project root)
   npm run dev
Or start backend and client separately:
   npm run server
   npm run client
